# Drum samples — attribution

The audio files in this folder are a curated subset (21 unmodified one-shot
samples, selected from ~6,000) of **DRSKit 2** by the DrumGizmo project —
Lars Muldjord and Bent Bisballe Nyeng, with the drum kit provided by
Jes Eiler of DRSDrums (drskit.dk).

- Source: https://drumgizmo.org/kits/ (via the stereo-per-mic SFZ port at
  https://github.com/sfzinstruments/DrumGizmo.DRSKit)
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
  https://creativecommons.org/licenses/by/4.0/
- Changes: none to the audio itself; files were selected and renamed
  (e.g. `8-Kdrum_without_contact-Kdrum_back.flac` → `kick_1.flac`).

This attribution must ship with the app (About box / credits) as long as
these samples are bundled.
